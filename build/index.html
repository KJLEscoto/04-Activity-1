User Experience

The website itself was extremely cluttered. The text, images, videos, and overall content is unorganized. Cluttered website can be overwhelming and frustrating for users, as it can make it difficult for them to find the information they need. It should be easy to use, accessible, visually appealing, and optimized for various devices and screen sizes to provide a consistent experience across all platforms.

Additionally, there must be a clear and user-friendly navigation system implemented on the website to enable visitors to effortlessly locate and retrieve the content they are seeking.

Content

As I explore through the website, the focus of it is primarily on topics related to technology, gadgets, and various types of transportation such as jeeps, scooters, cars, and etc. However, the images were stacked up with each other, having unclear width and height of all the elements presented. The content in this website does provides some information about the purpose of the site, but it is not adequately structured.

Therefore, to sustain the visitors' interest and keep them informed, it is necessary to update and refresh the content regularly.

Grid Layout

The grid layout is not consistent throughout the website. The layout was not visually balanced, with uneven distribution of content across the pages. Also, the grid was not designed to be responsive which it cannot adapt to different screen sizes and devices. Additionally, having a well-designed layout, it helps to establish a clear visual hierarchy, with important content positioned in prominent areas of the page

Moreover, the grid should be designed using appropriate proportions, with the size and spacing of elements carefully considered to create a pleasing visual aesthetic.
